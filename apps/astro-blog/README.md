# Astro Blog

`estrivault-blog-app` の描画層を Astro で提供するアプリです。

## Commands

```bash
pnpm --filter astro-blog dev
pnpm --filter astro-blog build
pnpm --filter astro-blog preview
pnpm --filter astro-blog check
pnpm --filter astro-blog test:e2e
```

## Routes

- `/`
- `/[page]`
- `/post/[slug]`
- `/category/[category]/[page]`
- `/tag/[tag]/[page]`
- `/sitemap.xml`
- `/llms.txt`
- `/llms-full.txt`
